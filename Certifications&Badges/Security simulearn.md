
## AWS SimuLearn – Core Security & Access Management

I completed an AWS SimuLearn scenario where I helped a fictional stock exchange strengthen its security by limiting support engineers’ access to only the permissions required for their roles. I designed a role‑based access management solution and then implemented it in a live AWS Console environment.

I worked with IAM users, roles, groups, and policies while applying least‑privilege principles and reviewing the AWS Shared Responsibility Model. I also gained hands‑on experience with IAM, Amazon EC2, and Amazon RDS as I validated the solution end‑to‑end. This simulation gave me practical experience combining real‑world problem‑solving with guided technical execution to reinforce secure access management best practices on AWS.
